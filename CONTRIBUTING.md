# Come contribuire

Grazie per l'interesse nel progetto! Ogni contributo è benvenuto.

## Segnalare un bug

Apri una **Issue** su GitHub descrivendo:
- Cosa hai fatto
- Cosa ti aspettavi che accadesse
- Cosa è accaduto invece
- Versione PHP e browser utilizzati

## Proporre una funzionalità

Apri una **Issue** con il tag `enhancement` e descrivi la funzionalità desiderata.

## Pull Request

1. Fai il **fork** del repository
2. Crea un branch descrittivo: `git checkout -b feature/nome-funzionalita`
3. Fai le modifiche e committa: `git commit -m "Aggiunge: descrizione"`
4. Fai il push: `git push origin feature/nome-funzionalita`
5. Apri una **Pull Request** verso il branch `main`

## Linee guida

- Mantieni il codice in un unico file PHP per vista (niente framework)
- Commenta le modifiche significative
- Non includere dati personali reali nei file di esempio
- Testa su PHP ≥ 7.4
